# LinkedIn Carousel

This folder contains a short carousel version of the project story for LinkedIn.

## Files

- `slide-01.png` through `slide-08.png`: upload these to LinkedIn as carousel images.
- `diagnostic-imaging-reconciliation-linkedin-carousel.pptx`: editable PowerPoint source.
- `deck-montage.webp`: quick visual preview.
- `linkedin-post-copy.md`: suggested caption text.

## Recommended Posting Order

1. Title and positioning
2. The workflow problem
3. Serverless event flow
4. AWS services used
5. Reconciliation rules
6. Failure handling
7. Architecture tradeoffs
8. Lessons learned and privacy boundary

The carousel intentionally avoids real PHI, workplace screenshots, internal logs, server names, credentials, and production compliance claims.
